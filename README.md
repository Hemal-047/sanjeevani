# Sanjeevani

**Private health cognition → trustworthy public action. Powered by Venice AI.**

A multi-agent health intelligence system for the Synthesis hackathon (Venice AI track). Sanjeevani privately analyzes medical records through Venice's zero-retention AI, discovers health patterns across documents, and publishes anonymous onchain health attestations for clinical trial matching and data marketplace.

## Project Structure

```
/backend    — Node.js Express API server
/frontend   — React app (Vite + TailwindCSS)
/contracts  — Solidity smart contracts (Hardhat, Base Sepolia)
```

## Getting Started

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev

# Contracts
cd contracts && npm install && npx hardhat compile
```
